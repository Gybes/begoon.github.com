Описание файлов, входящих в данный релиз эмулятора

Начиная с версии 0.5 архив эмулятора НЕ содержит программ для него. Архивы с всеми программами распростараняются отдельно. Архивы с программами надо распаковать в каталоги TAPE16 или TAPE32. Для этого достаточно скопировать эти архивы (они поставляются как файлы selt-extract) и запустить их.

PK86EMUL.EXE Исполняемый модуль эмулятора  
PK86DBG.EXE Исполняемый модуль эмулятора с отладчиком  
KEYBOARD.HLP Файл раскладки клавиатуры для режима 'РК'  
FONT16.FNT Файл с 8x16 шрифтом эмулятора  
FONT8.FNT Файл с 8x8 шрифтом эмулятора  
MON16.BIN Монитор для режима 16кб  
MON32.BIN Монитор для режима 32кб  
FILE_ID.DIZ Short description  
Каталог DOCS\ Файлы документации  
EMUL.TXT Описание эмулятора  
DEBUG.TXT Описание встроенного отладчика  
FILES.TXT То, что Вы сейчас читаете  
HISTORY.TXT Хронология развития эмулятора  
TODO.TXT Что планируется еще сделать  
INTERNET.TXT Internet ресурсы эмулятора  
RSS4BETA.TXT Описание формата snapshot'ов RSS, редакция 4 beta
